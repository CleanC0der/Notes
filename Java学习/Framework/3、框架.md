3、框架
2021年7月13日
9:19

框架是一个舞台，一个模板

模板：
1.  规定好了一些条款，内容。
2.  加入自己的东西
框架是一个模块
1.  框架中定义好了一些功能，这些功能是可用的。
2.  可以加入项目中自己的功能，这些功能可以利用框架写好的功能

框架是一个软件，半成品的软件，定义好了一些基础功能，需要加入你的功能就是完整的。基础功能是可重复使用的，可升级的。

框架特点：
1.  框架一般不是全能的，不能做所有事情。
2.  框架是针对某一个领域有效。特长在某一个方面，比如mybatis做数据库强，但不能做其他的。
3.  框架是一个软件
