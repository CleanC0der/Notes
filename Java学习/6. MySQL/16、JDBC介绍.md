JDBC介绍
根据Oracle的声明，JDBC是一个注册了商标的术语，而并非Java DataBase Connectivity的首字母缩写。对它的命名体现了ODBC的致敬，后者是微软开创的标准数据库API，并因此而并入了SQL标准中。

JDBC接口的class文件在java.sql里面
各大数据库厂家需要面向这个接口写代码
我们用哪个数据库，就从哪个数据库官网下包
