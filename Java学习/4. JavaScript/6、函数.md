**函数**

- function 函数名(形参1,形参2,形参3, …){}
- 其他和Java一样
- **函数调用（普通函数调用）**
  - functionName();//返回就是return的东西
  - 函数中的代码将在其他代码调用该函数时执行：
  - 当事件发生时（当用户点击按钮时）
  - 当 JavaScript 代码调用时
  - 自调用
  - **不使用 () 访问函数将返回函数声明而不是函数结果。**
- **函数调用（当构造函数调用）**
  - new functionName();//这样函数体的return无效，直接返回一个object对象
